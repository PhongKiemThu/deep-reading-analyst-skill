# Deep Reading Analyst - Claude AI 深度阅读技能

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/version-2.0.0-blue.svg)](https://github.com/yourusername/deep-reading-analyst)

一个专业的 Claude AI 技能包，通过系统化的思维框架将表面阅读转化为深度学习。

[English](README.md)

## 🌟 核心特性

通过 **10+ 种思维框架** 提升阅读体验：

### 快速分析（15 分钟）
- 📋 **SCQA 框架** - 结构化思维（情境-冲突-问题-答案）
- 🔍 **5W2H 分析** - 完整性检查（What、Why、Who、When、Where、How、How much）

### 标准分析（30 分钟）
- 🎯 **批判性思维** - 识别逻辑漏洞和论证缺陷
- 🔄 **逆向思维** - 风险识别和失败模式分析

### 深度分析（60 分钟）
- 🧠 **多元思维模型** - 多学科视角（物理学、生物学、心理学、经济学）
- ⚡ **第一性原理** - 剥离假设，回归本质
- 🔗 **系统思维** - 建立知识关联，洞察全局
- 🎨 **六顶思考帽** - 多角度全面评估

### 研究级别（120 分钟+）
- 📊 **跨源对比** - 多文章综合分析和整合洞察

## 🎯 技能功能

本技能提供系统化的 5 步分析流程：

1. **初始化** - 设定目标和深度级别
2. **结构理解** - 提取文章结构和核心概念
3. **思维模型** - 运用多种框架（4 个深度级别可选）
4. **综合输出** - 基于目标生成定制化内容
5. **知识激活** - 创建可执行的下一步行动

## 📊 分析深度级别

根据需求选择合适的级别：

| 级别 | 时间 | 适用场景 |
|------|------|----------|
| **Level 1（快速）** | 15 分钟 | 快速了解核心观点 |
| **Level 2（标准）** | 30 分钟 | 日常学习和理解 |
| **Level 3（深度）** | 60 分钟 | 重要内容深度分析 |
| **Level 4（研究）** | 120 分钟+ | 学术研究和跨源对比 |

## 🚀 安装方法

### 前置要求

- Claude 桌面应用或 Claude 网页版 (claude.ai)
- Claude Pro 订阅（推荐，以获得最佳体验）

### 安装步骤

1. **下载** 技能文件：[deep-reading-analyst.skill](deep-reading-analyst.skill)

2. **导入到 Claude：**

   **桌面应用：**
   - 打开 Claude 桌面应用
   - 点击右上角头像
   - 选择 Settings → Skills
   - 点击 "Import Skill"
   - 选择下载的 `.skill` 文件

   **网页版：**
   - 访问 [claude.ai](https://claude.ai)
   - 点击右下角设置图标
   - 进入 "Skills" 标签页
   - 点击 "Import" 按钮
   - 上传 `.skill` 文件

3. **开始使用！** 当你要求 Claude 分析文章时，技能会自动触发。

## 💡 使用指南

### 基础用法

直接提供文章并要求分析：

```
请深度分析这篇文章：[粘贴内容或 URL]
```

### 指定分析深度

```
请用 Level 3（深度）模式分析这篇关于软件工程的文章
```

### 请求特定思维框架

你可以明确请求任何框架：

```
请用批判性思维分析这篇文章的论证质量
请用第一性原理拆解这个概念
请用系统思维分析这些要素之间的关系
请用六顶思考帽全面评估这个观点
```

### 请求特定输出格式

```
请分析这篇文章并提供：
1. 学习笔记（给自己看的）
2. 实践应用计划
3. 概念地图
```

## 💡 技巧和最佳实践

### ✅ 推荐做法

1. **明确目的** - 告诉 Claude 你阅读的目的（学习/解决问题/写作参考）
2. **分段消化** - 对于超长文章，可以先分析部分章节
3. **主动提问** - 在分析过程中主动提出疑问
4. **实践验证** - 要求生成可执行的行动计划
5. **建立连接** - 要求关联到你已知的知识

### ❌ 避免做法

1. 不要一次性分析过多文章（建议 ≤3 篇）
2. 不要期望技能代替阅读（它是辅助工具）
3. 不要忽视技能提出的批判性问题
4. 不要只收集笔记不实践

- **论证分析** - 详细的论证质量评估

- **分享文案** - 给别人看的精炼总结
- **教学大纲** - 用于向他人解释的内容结构

## ❓ 常见问题

**Q：技能能否直接读取网页链接？**
A：可以！Claude 可以使用 web_fetch 工具获取网页内容。

**Q：分析一篇文章需要多久？**
A：取决于深度级别，从 15 分钟（快速）到 120 分钟以上（研究级）不等。

**Q：我可以要求只使用某个思维模型吗？**
A：当然可以！明确告诉 Claude 你想用哪个模型即可。

**Q：技能能分析非英文内容吗？**
A：可以！支持中文、英文等多种语言。

**Q：如何让分析更符合我的需求？**
A：提供更多上下文信息，如你的背景、阅读目的、关注重点等。

**Q：版本 2.0 有什么新功能？**
A：新增 4 个主要框架：SCQA（麦肯锡）、5W2H 分析、多元思维模型（30+ 个模型）和逆向思维。框架总数从 6 个增加到 10+ 个。
